**Tic Tac Toe: Adaptive Bot System** is a modern strategic duel  
where artificial intelligence evolves based on your performance.  
Win once → bot enters **AGGRESSIVE MODE**  
Draw repeatedly → system recalibrates for balance  
Every match shapes the next.
